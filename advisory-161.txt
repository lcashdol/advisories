business directory, staff directory or church directory.
Vulnerability:
Line 320 contains unfiltered user input for the search field being sent
directly via echo back to the users browser via the ’s’ variable. In file
includes/admin/pages/manage.php 320: <input type="search" id="entry-search-
input" name="s" value="<?php if ( isset( $_GET['s'] ) && ! empty( $_GET['s'] )
) echo $_GET['s'] ; ?>" />
